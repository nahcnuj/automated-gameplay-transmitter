# Examples

This directory contains example apps built with `automated-gameplay-transmitter`.

## Requirements

- [Node.js](https://nodejs.org/) (with npm)

## Setup

```sh
npm install
```

## Development

Start the Vite dev server:

```sh
npm run dev
```

Open http://localhost:5173/ in your browser.

## Build

Produce a production build:

```sh
npm run build
```

Output is written to `dist/`.

## Type check

```sh
npm run typecheck
```
